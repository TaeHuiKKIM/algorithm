# Algorithm

BaekjoonHub로 자동 push되는 백준 풀이 저장소입니다. C/C++ 중심으로 Bronze, Silver, Gold 단계의 문제 풀이가 정리되어 있습니다.

## Contents

- `백준/Bronze`: 기초 입출력, 조건문, 재귀 문제
- `백준/Silver`: 자료구조, 정렬, 그리디, DP, 그래프 기초
- `백준/Gold`: 분할정복, 행렬 제곱, 표기식 처리 등

현재 저장소에는 약 70개의 풀이 파일이 포함되어 있습니다.

## Notes

각 문제 폴더는 BaekjoonHub가 생성한 구조를 따릅니다. 풀이 코드는 문제별 폴더 안의 소스 파일에서 확인할 수 있습니다.
